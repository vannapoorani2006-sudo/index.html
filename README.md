## 🚀 Deployment

This site is deployed using **GitHub Pages**:

1. Repository contains a single `index.html` file at the root
2. GitHub Pages is enabled via **Settings → Pages → Deploy from a branch → main → / (root)**
3. The site is live at the URL shown under Settings → Pages once deployment finishes

To update the site, simply replace `index.html` in this repository with a new version — GitHub Pages redeploys automatically.
